# 🌐 Web UI Quick Start

## Launch the Web Interface

```bash
cd web-ui
./start.sh
```

Or manually:

```bash
cd web-ui
pip3 install -r requirements.txt
python3 app.py
```

Then open **http://localhost:5000** in your browser.

## What You Can Do

✅ **Configure tenant settings** with a beautiful form interface  
✅ **Add/remove products** dynamically  
✅ **Save params.txt** automatically in params directory  
✅ **Run setup scripts** with one click  
✅ **Validate configuration** instantly  
✅ **View product summary** in real-time  

## Features

- All operations modify the actual `params.txt` file in the params directory
- Scripts execute from the scripts directory
- Results are generated in the results directory
- Real-time output displayed in the browser
- No manual file editing required!

Enjoy! 🎉
